# Decidoo

Decidoo is an AI-assisted food decision app designed to answer one simple question: **What should I eat?**

## MVP

- Fast food recommendation flow
- Mood, budget and meal-time preferences
- Local mock recommendation engine
- Clean Flutter interface
- Ready for location, weather and AI integrations

## Run

```bash
flutter pub get
flutter run
```

## Product direction

The first release focuses only on food decisions. Future versions may use location, weather, time, dietary preferences and recommendation history to personalize results.
