# Decidoo

Decidoo is a global AI decision engine for food.

## Revenue Architecture

### Consumer
- Free tier
- Premium AI subscription
- Loyalty rewards
- Gift cards

### Restaurant
- Subscription plans
- Sponsored placements (clearly labeled)
- Campaign manager
- Analytics dashboard

### Commerce
- Order commission
- Reservation commission

### Enterprise
- Business accounts
- Employee meal programs

### Platform
- Decision API
- Partner integrations

### Insights
- Anonymous market analytics for restaurants

## Rollout
Phase 1: Free + Premium + Restaurant subscriptions
Phase 2: Sponsored listings + Commissions + Loyalty
Phase 3: Business + API + Analytics

## Principle
Trust first. Commercial placements are always clearly disclosed.
